# B2S API Reference

## Hiro API Endpoints
- Balances: GET /extended/v1/address/{address}/balances
- Transactions: GET /extended/v1/address/{address}/transactions
- NFT Holdings: GET /extended/v1/tokens/nft/holdings
- FT Holders: GET /extended/v1/tokens/ft/holders

## Agent API
- POST /api/agent — Send message to B2S AI Agent
- GET /api/agent — Get agent status
